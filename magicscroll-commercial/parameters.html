Parameters for Magic Scroll™
Parameter Default Options Description
speed 5000 ms Change the scroll speed (0 - manual)
duration 1000 ms Duration of effect
loop continue restart/continue Restart scroll after last image (Continue to next image or scroll all the way back)
width auto 'auto' or numeric Scroll width (pixels)
height auto 'auto' or numeric Scroll height (pixels)
item-width auto 'auto' or numeric Scroll item width (pixels)
item-height auto 'auto' or numeric Scroll item height (pixels)
items 3 numeric Items to show (0 - manual)
step 3 numeric Scroll step
arrows inside false/inside/outside Show arrows
arrows-opacity 60 numeric Opacity of arrows
arrows-hover-opacity 100 numeric Opacity of arrows on mouse over
direction right top/right/bottom/left Direction of scroll
slider false false/top/right/bottom/left Slider postion
slider-size 10% numeric or percent Slider size
item-tag A A/IMG/DIV HTML tag that will be recognized as item element
